# In mind tic tac toe for Android devices

It is my developmental project to learn to code for Android Devices. Since efficient script languages, like Kotlin, are now avaliable to natively code for android Devices, fast writing of android apps became possible.

Play 2D 3x3 Tic Tac Toe on Android in Yours own brain memory with friend !!! and have a smartphone as a referee. 
You chan choose to play with other Human or with AI. You are Human 1 (H1) and Your friend is Human 2 (H2) or AI (A1). 
Choose Row and Column and place mark by pressing "Set mark in Row and Column on board in mind". 
Mistakes numbers and elapsed times for you and your friend are displayed below. 
In case no body gets 3 in row, the winner is the person with the lowest number of mistakes, and in case of another tight a shorter elapsed time is a winner.
